Ccsim UI

选择一种方式打开访客端或客服端测试页面

NPM — 本地开发（vite dev server）
访客 visitor-npm.html import from src › 客服 agent-npm.html import from src ›
UMD — script 标签加载
访客 visitor-umd.html dist/ccsim-ui.umd.js › 客服 agent-umd.html dist/ccsim-ui.umd.js ›
ES Module — import map
访客 visitor-esm.html dist/ccsim-ui.es.js › 客服 agent-esm.html dist/ccsim-ui.es.js ›
前置条件：后端服务 go run . + 前端开发服务 npm run dev（仅 npm 方式需要）。 UMD / ESM 方式需先 npm run build 生成 dist 产物。
